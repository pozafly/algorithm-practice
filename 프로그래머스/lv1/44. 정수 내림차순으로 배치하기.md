## 나의 답

```js
function solution(n) {
    return Number([...String(n)].sort((a, b) => b - a).join(''));
}
```

- 맞춤
- 문자열로 소팅함.

## 다른 답

```js
function solution(n) {
    //숫자가 분명히 더 빠름
    var nums =[];
    do{
        nums.push(n%10);
        n=Math.floor(n/10);
    } while(n>0)

    return nums.sort((a,b)=>b-a).join('')*1;
    //문자는 느림
    return (n+"").split('').sort((a,b)=>b-a).join('')*1;
}
```

- 숫자 형태로 계산해서 푸는 것이 더 빠르다고 한다.
- nums라는 배열에 차례로 집어넣었다. 숫자를.
- 숫자로 푸는 방법을 알아둘 필요는 있을듯.
